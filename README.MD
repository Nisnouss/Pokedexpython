# Projet d'évaluation - Pokedex

## Objectif du projet

Le but de ce projet est de créer un répertoire à pokemon, permettant de voir leur image, ainsi que leur caractéristiques. L'utilisateur peut également remplir un formulaire afin d'ajouter un autre pokemon au répertoire.

## Composantes du projet
Le répertoire se présente sous forme d'une listbox tkinter listant les pokemons. En sélectionnant un pokemon et en appuyant sur un bouton, les caractéristiques ainsi que l'image du pokemon s'affichent sur une nouvelle fenêtre. 

Il y a également un formulaire permettant de rajouter un pokemon au répertoire, dans lequel il suffit d'introduire les informations demandées dans les champs de saisies spécifiques, et valider par un bouton.

Enfin, deux bouton permettent la suppression ou la modification d'un des éléments du menu.
